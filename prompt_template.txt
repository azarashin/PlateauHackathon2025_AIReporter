【出力形式】
分析結果は次のようなjson形式で出力してください。
説明文は一切書かないでください。
出力される文字列は"{"で始まり、"}"で必ず終わります。

{
    "abstract": (要旨),
    "sections": [
        { "title":(分析タイトル名), 
            [
                {"type": "text", "content": "(説明文段落)"}, 
                {"type": "image", "content": "(画像への相対パス)", "title": "(画像のタイトル)"}, 
                {"type": "table", "content": "(表を示すjson データ。行のリストになっていなければならない)", "title": "(表のタイトル)}]}
            ]
        }, 
        ...
    ]
}

text 要素は最低一つ必要です。
image, table を使用するかどうかは任意です。

もしtable を使用する場合、table 要素の下にtext 要素を追加し、このtext 要素によってtable 要素の内容に対する見解・考察を必ず述べるようにしてください。

決して"```json" を含めないでください。
「レポートは以下の通りです。」や「JSON形式のデータを作成しました。」といったメッセージなど、json データの文字列を絶対に入れないようにしてください。
得られる出力は"{" で始まり、"}" で終わるはずです。
