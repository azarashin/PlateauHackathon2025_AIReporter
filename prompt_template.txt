出力される文字列は"{"で始まり、"}"で必ず終わります。

{
    "abstract": (要旨),
    "sections": [
        { "title":(分析タイトル名), 
            [
                {"type": "text", "content": "(説明文段落)"},

text 要素は最低一つ必要です。
image, table を使用するかどうかは任意です。

もしtable を使用する場合、table 要素の下にtext 要素を追加し、このtext 要素によってtable 要素の内容に対する見解・考察を必ず述べるようにしてください。
